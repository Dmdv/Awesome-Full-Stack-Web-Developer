This shared folder contains the full set of slides and videos for the Algorithmic Toolbox course of the Data Structures and Algorithms Specialization at Coursera. One of the main features of this class (and the whole specialization) is programming assignments. In such assignments, your goal is to implement an algorithm solving a given computational problem in less than a few seconds, even on large datasets. Solving such programming assignments will give you a much deeper understanding of algorithms. You will also gain an invaluable experience in solving programming problems similar to those given during interviews at high-tech companies as well as testing and debugging your solutions.
